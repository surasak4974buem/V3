2. สรุปการอัพเดทล่าสุด (V3 — 13/06/2569)

🔑
LINE Token + Group ID บันทึกผ่าน UI (Admin-only)

ไม่ต้อง hardcode credentials ใน code.gs อีกต่อไป — Admin กรอก Token และ Group ID ในหน้าตั้งค่าได้เลย ระบบบันทึกลง localStorage เหมือนกับ GAS URL

🛡️
Admin-only Permission สำหรับการตั้งค่าระบบ

เพิ่ม permission check — เฉพาะ Admin เท่านั้นที่กด "บันทึกการตั้งค่า" ได้ (GAS URL, LINE Token, รหัสผ่าน) ป้องกัน User ทั่วไปแก้ไขค่าสำคัญ

🌐
แก้ CORS "Failed to fetch"

เปลี่ยน Content-Type จาก application/json เป็น text/plain ในทุก POST request ไป GAS — ป้องกัน CORS preflight ที่ GAS ไม่รองรับ

🔔
LINE แจ้งเตือนทำงานอัตโนมัติ

เปิด Toggle + กรอก Token + Group ID → ระบบส่ง LINE Flex Message ทุกครั้งที่มีรายการ Receive/Issue/Borrow/Return/Adjust

📋
code.gs ใหม่ — ไม่มี hardcode credentials

GAS รับ token/groupId จาก client request แทนค่า hardcode เพิ่มความยืดหยุ่น ช่างเทคนิคแต่ละคนตั้งค่า LINE ของตัวเองได้

🐛
แก้ null reference crashes ทั้งหมด

เพิ่ม null guards ใน handleTxTypeChange, updateDashboard, renderDashboardAlertsTable, drawDashboardCharts, modal-sc-stock, memo-chk-other-text

🗃️
DATA_VERSION gate "v3-clean-2026-06-13"

ล้าง localStorage เก่าอัตโนมัติเมื่อเปิดระบบครั้งแรก ป้องกันข้อมูล mock จากเวอร์ชันก่อนหน้ารบกวน

1

สร้าง LINE OA Channel

เข้า developers.line.biz → Create a Provider → Create a new channel → Messaging API

2

ออก Channel Access Token

แท็บ Messaging API → Channel access token → กด Issue → Copy token

3

ตั้ง Webhook URL

Webhook URL = GAS Web App URL ที่ deploy ไว้ → Verify → ควรขึ้น Success

4

หา User ID / Group ID

Add LINE OA เป็นเพื่อน → ส่งข้อความหา bot → ดู ID ที่บันทึกในชีต LINE_IDs ของ Google Sheets

5

กรอกในระบบ

Login Admin → หน้าตั้งค่า → กรอก LINE Token + Group ID → บันทึก → ทดสอบ

5. แดชบอร์ด (Dashboard)

หน้าแรกหลัง Login แสดงภาพรวมสถานะคลังอะไหล่ทั้งหมด

5.1 การ์ดสรุป (KPI Cards)

การ์ดแสดงข้อมูลสีเตือน
รายการอะไหล่ทั้งหมดจำนวน SKU ในคลังปกติ: ฟ้า
มูลค่าคลังรวมunitPrice × inStock ทุกรายการปกติ: ม่วง
รายการสต๊อกต่ำจำนวน SKU ที่ inStock ≤ reorderPointแดงนีออน เมื่อมี
ใกล้หมดอายุSKU ที่ expiryDate ≤ 90 วันส้มนีออน เมื่อมี

5.2 ตารางแจ้งเตือน

แสดงรายการที่ สต๊อกต่ำกว่า Reorder Point หรือ หมดอายุภายใน 90 วัน พร้อมปุ่มลัด "เบิก/รับ" ได้เลย

5.3 กราฟ

  • Transaction History Chart — แสดงจำนวนรายการ Receive/Issue/Borrow/Return รายวัน
  • ABC Analysis Chart — แสดงสัดส่วนมูลค่าอะไหล่ตามคลาส A/B/C

5.4 Quick Actions

ปุ่มลัดจากแดชบอร์ด: รับเข้าคลัง | เบิกอะไหล่ | ซิงก์คลาวด์ | ดึงข้อมูลจากคลาวด์

6.3 ABC Classification

ระบบคำนวณ ABC อัตโนมัติจาก มูลค่ารวม (unitPrice × inStock):

  • Class A — มูลค่าสูง top 20% ของรายการ (ควบคุมเข้มงวด)
  • Class B — มูลค่าปานกลาง 30% ถัดไป
  • Class C — มูลค่าต่ำ 50% ที่เหลือ

7. บันทึกรายการ (Transactions)

หัวใจหลักของระบบ — บันทึกการเคลื่อนไหวของอะไหล่ทุกประเภท พร้อมส่ง LINE แจ้งเตือนอัตโนมัติ

ประเภทความหมายผล (inStock)สี LINE
Receive ➕รับอะไหล่เข้าคลัง+ เพิ่มขึ้นเขียว
Issue 📤เบิกจ่ายอะไหล่ใช้งาน− ลดลงแดง
Borrow 🔍ยืมอะไหล่ทดสอบ (ต้องคืน)− ลดลง (ชั่วคราว)ส้ม
Return 📥คืนอะไหล่กลับคลัง+ เพิ่มขึ้นเขียวอมฟ้า
Adjust ⚙️ปรับยอดสต๊อก (ตรวจนับ)ตั้งค่าใหม่ม่วง

7.1 ขั้นตอนบันทึกรายการ

1

เลือกประเภทรายการ

แท็บ Receive / Issue / Borrow / Return / Adjust ที่ด้านบนฟอร์ม

2

ค้นหาและเลือกอะไหล่

พิมพ์ชื่อหรือรหัสในช่องค้นหา → เลือกจากรายการที่ปรากฏ

3

กรอกรายละเอียด

จำนวน, เอกสารอ้างอิง, เลขซ่อม (บังคับสำหรับ Issue/Borrow), ซีเรียลเครื่อง (บังคับ), แผนก, หมายเหตุ

4

กด "บันทึกรายการ"

ระบบบันทึก → อัปเดต inStock → ส่ง LINE แจ้งเตือน → ซิงก์ Google Sheets

15. การแก้ไขปัญหาที่พบบ่อย

❌ "Failed to fetch" เมื่อทดสอบ LINE
  • ตรวจสอบว่ากรอก GAS URL ถูกต้องและกด "บันทึกการตั้งค่า" แล้ว
  • GAS ต้อง Deploy ด้วย "Anyone" access
❌ "Authentication failed" จาก LINE
  • Token หมดอายุหรือถูก revoke — ไป LINE Developers Console ออก Token ใหม่
  • ตรวจว่า copy token ครบ ไม่มีช่องว่างต้น/ท้าย
❌ "Failed to send messages" จาก LINE
  • Group ID / User ID ไม่ถูกต้อง
  • User ยังไม่ได้ Add LINE OA เป็นเพื่อน
  • ส่งข้อความหา bot แล้วดู ID ที่ถูกต้องจากชีต LINE_IDs ใน Google Sheets
❌ ข้อมูลหายหลัง Refresh
  • ข้อมูลอยู่ใน localStorage — ปกติไม่หาย ยกเว้น Clear browser data
  • กด "ดึงข้อมูลจากคลาวด์" เพื่อกู้ข้อมูลจาก Google Sheets
❌ ปุ่ม "บันทึกการตั้งค่า" ไม่ทำงาน
  • ต้อง Login ด้วย Account ที่มีสิทธิ์ Admin เท่านั้น
❌ GAS ตอบกลับ {} ว่างเปล่า
  • Sheet ยังไม่มีชื่อชีต (Parts/Transactions) — กด "ซิงก์คลาวด์" เพื่อสร้าง Sheet ก่อน
  • หรือรัน resetDatabase() ใน Apps Script Editor แล้วซิงก์ใหม่
